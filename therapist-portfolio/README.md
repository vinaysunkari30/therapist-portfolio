Therapist Portfolio | Dr. Maya Reynolds

A high-end, responsive portfolio website built for Dr. Maya Reynolds, a Clinical Psychologist specializing in Anxiety, Trauma, and Burnout. The site is designed with a calming aesthetic, featuring luxury image galleries and professional treatment descriptions.

Tech StackFramework:

Next.js 16 (App Router)
Library: React 19
Styling: Tailwind CSS 4 (using PostCSS)
Icons: React Icons
Linting: ESLint

Key FeaturesResponsive Grid System: 
Custom layouts using grid-cols-1 for mobile and md:grid-cols-2 for desktop.Luxury Hover Effects: Implementation of group-hover and relative positioning for "floating" image text overlays.

Modern Typography: Clean headings and body text optimized for readability.Performance: Optimized images and fast page loads via Next.js.

1. Clone the repository:
  git clone git clone [https://github.com/vinaysunkari30/therapist-portfolio.git](https://github.com/vinaysunkari30/therapist-portfolio.git)
  cd therapist-portfolio

2. Install dependencies
  npm install

3. Run the development server
  npm run dev

Open http://localhost:3000 with your browser to see the result.
