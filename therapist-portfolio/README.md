**Therapist Portfolio | Dr. Maya Reynolds**

A high-end, responsive portfolio website built for Dr. Maya Reynolds, a Clinical Psychologist specializing in Anxiety, Trauma, and Burnout. The site is designed with a calming aesthetic, featuring luxury image galleries and professional treatment descriptions.

**Tech StackFramework:**

Next.js 16 (App Router)
Library: React 19
Styling: Tailwind CSS 4 (using PostCSS)
Icons: React Icons
Linting: ESLint

1. **Clone the repository:**
  git clone https://github.com/vinaysunkari30/therapist-portfolio.git

### Navigate to Project
  cd therapist-portfolio

3. **Install dependencies**
  npm install

4. Run the development server
  npm run dev

Open http://localhost:3000 with your browser to see the result.
