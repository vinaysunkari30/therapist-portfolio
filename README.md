# Therapist Portfolio

A modern and responsive therapist portfolio website built with Next.js. 
The website showcases professional information, services, experience, contact details, and provides an engaging user experience for potential clients.

## Features

- Responsive design for all devices
- Modern and clean user interface
- Therapist profile and introduction
- Services section
- About section
- Contact section

## 🛠️ Tech Stack

### Frontend
- Next.js
- React.js
- JavaScript
- HTML5
- CSS3
- Tailwind.css

### Deployment
- Vercel

## Installation

### Clone Repository

```bash
git clone https://github.com/vinaysunkari30/therapist-portfolio.git
```

### Navigate to Project

```bash
cd therapist-portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will run at:

```text
http://localhost:3000
```

## Build for Production

```bash
npm run build
```

## Start Production Server

```bash
npm start
```

## Purpose

This project was developed to provide therapists with a professional online presence where they can showcase their 
expertise, services, and allow potential clients to connect with them easily.
